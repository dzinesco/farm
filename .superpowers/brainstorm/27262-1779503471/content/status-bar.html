Status bar & coordinate display

Where does info live, and how is it organized?

A

Ruler-corner readout only

No bottom status bar. Coordinate info lives in the corner where the rulers meet (like AutoCAD). Tool name and wall count shown in a minimal top-left floating chip. Clean, minimal.

B

Single-line minimal bar

A single 24px bottom bar. Left: tool name + wall count. Center: active dimension during draw operations (live). Right: zoom level + snap/ortho indicators. Nothing else.

Floating HUD overlay

No status bar at all. Coordinates show at cursor during operations. Tool name in top-left floating chip. Snap/ortho in top-right floating chip. Everything floats, nothing is static.

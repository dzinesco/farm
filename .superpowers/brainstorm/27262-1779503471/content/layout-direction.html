Overall layout direction

How should the main editor chrome feel?

A

Dense CAD Bar

Horizontal top toolbar, everything visible. All tools in your face, information-dense. Like AutoCAD or SketchUp.

B

Minimal Dark Bar

Top toolbar shows only what's relevant per active tool. Clean, dark, lots of breathing room in the canvas. Like Figma or Linear.

C

Floating Toolbar

No horizontal bar. Tools float as a compact pill top-left, properties slide in from right, full canvas priority. Like some game editors.

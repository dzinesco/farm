Command palette & keyboard shortcuts

How do you invoke and discover commands?

A

Cmd+K command palette only

Press Cmd+K → overlay search box. Type "wall", "snap", "undo", "3D", "plan" to find commands. Shortcuts shown inline in results. No visible shortcut legend panel — just discoverable via Cmd+K. Clean and minimal.

B

Cmd+K + floating ? shortcut panel

Cmd+K for command palette. Press ? to show a floating shortcut legend panel (semi-transparent, dismissable). Shows all shortcuts grouped by category — like Figma's help overlay.

C

No command palette, shortcuts in toolbar

No Cmd+K. Shortcuts shown directly in the tool buttons (e.g., toolbar shows "Select (V)", "Pan (H)"). Persistent keyboard shortcut panel accessible via "?" button in the tools panel. More like classic CAD.

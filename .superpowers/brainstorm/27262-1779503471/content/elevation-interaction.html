Elevation view interaction model

Click panels in elevation view to...

A

Fully interactive

Click panel → selected (cyan outline). Drag to move. Double-click stud → edit position. Direct manipulation like Figma — plan and elevation both as editors.

B

Read-only visualization

Elevation shows layer stack and framing as a diagram. Clicking selects the whole wall. Editing happens only in plan view and 3D inspector. Like reading a drawing, not editing it.

C

Click → inspect popover

Click panel → small popover with name, dimensions, layer list. Drag not supported. For details-on-demand rather than direct editing. Edits via right-click properties panel.

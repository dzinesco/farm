Multi-accent palette — interesting

Dark + cyan/pink/yellow/magenta accents

1

Color-coded by category

Each tool category gets a color. Select=cyan, Draw=pink, Pan=yellow, Edit=magenta. View modes get their own colors. Makes the UI immediately scannable — you know "pink = drawing tools".

2

Color-coded by panel type

Tools panel=cyan, Layers panel=pink, Properties panel=magenta, Snap/Grid=yellow. Each panel has its own accent so you instantly know which panel you're looking at.

3

Color-coded by state

Colors mark interactive states — active=cyan, hover=pink, selected=yellow, warning=magenta. More subtle, feels "alive" rather than categorically loud.
